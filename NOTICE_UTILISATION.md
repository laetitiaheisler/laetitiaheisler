# Notice d'utilisation — Site Laetitia Heisler

## 1. Modifier le site

### Modifier du texte
1. Allez sur https://github.com/laetitiaheisler/laetitiaheisler
2. Cliquez sur le fichier à modifier (ex: `index.html`)
3. Cliquez sur l'icône ✏️ (crayon) en haut à droite
4. Modifiez le texte
5. Scrollez en bas, écrivez un titre (ex: "Mise à jour texte")
6. Cliquez **"Commit changes"**

Le site se met à jour automatiquement en 1-2 minutes.

### Changer une photo
1. Allez sur https://github.com/laetitiaheisler/laetitiaheisler
2. Cliquez sur le dossier **`images/`**
3. Cliquez sur **"Add file"** → **"Upload files"**
4. Glissez votre nouvelle photo
5. **Important** : le nom du fichier doit être EXACTEMENT le même que l'ancien (sinon ça ne marchera pas)
6. Scrollez en bas, cliquez **"Commit changes"**

### Ajouter une nouvelle page
1. Allez sur https://github.com/laetitiaheisler/laetitiaheisler
2. Cliquez **"Add file"** → **"Create new file"**
3. Donnez un nom (ex: `expositions.html`)
4. Copiez le contenu d'une page existante (ex: `contact.html`) pour avoir la structure
5. Modifiez le titre et le contenu
6. **Commit changes"

## 3. Structure d'une page

```
<header>
  <nav>         ← MENU (ne pas toucher)
    <a href="workshops.html">Workshops</a>
    <a href="index.html">Works</a>
    ...
  </nav>
</header>

<main>
  <h1>titre</h1>
  <p>votre texte ici</p>
</main>

<footer>
  © Copyright 2026 | Laetitia Heisler
</footer>
```

---

## 4. Pour ajouter un lien dans le menu
1. Ouvrez TOUS les fichiers `.html`
2. Dans chaque `<nav>`, ajoutez : `<a href="ma-page.html">Ma Page</a>`
3. Créez le fichier `ma-page.html`
