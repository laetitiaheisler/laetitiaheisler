# Notice d'utilisation — Site Laetitia Heisler

## Modifier le site (le plus simple)

1. Allez sur **https://laetitiaheisler.com/admin**
2. Connectez-vous avec votre compte GitHub
3. Cliquez sur la section à modifier (À propos, Contact, Actualités, Œuvres)
4. Modifiez le texte dans les formulaires
5. Pour changer une photo : cliquez sur l'image, choisissez "Upload"
6. Cliquez **"Publish"** — le site se met à jour tout seul

## Pour les photos (première fois)

Il faut d'abord uploader vos 8 photos dans le dossier `images/` sur GitHub :

1. Allez sur https://github.com/laetitiaheisler/laetitiaheisler/tree/main/images
2. Cliquez **"Add file"** → **"Upload files"**
3. Glissez vos 8 photos
4. Scrollez en bas, cliquez **"Commit changes"**

### Noms de fichiers obligatoires
- `the-sleepy-menstruant.jpg`
- `life-and-death-bond.jpg`
- `at-the-wrong-place.jpg`
